# Change Log

All notable changes to the "FIDE Embedded Dashboard" extension will be documented in this file.

## [0.0.1] - 2026-02-11

### Added
- Initial release of FIDE Embedded Dashboard
- Dashboard webview with tabbed interface
- New Project page with board selection
- Support for 4 development boards:
  - STM32F4 Discovery
  - ESP32 DevKitC
  - Raspberry Pi Pico
  - Nordic nRF52840 DK
- Log Viewer page with real-time activity logs
- Backend API integration via HTTP
- Automatic workspace creation
- Project template download and expansion
- Configuration setting for backend URL

### Features
- Visual board selection with specifications display
- Project creation wizard
- Activity logging system
- Workspace file generation
- File tree expansion from backend templates
